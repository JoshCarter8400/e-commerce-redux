# E-Commerce Store with Redux

This is a MERN stack project that features on online store that allows a user to save items to their cart, checkout, and view all item descriptions. The whole site functions whether
they have an internet connection or not.

## Tools Used for building this Project

- React
- Redux
- React Router
- Apollo Server and Apollo Boost
- MongoDB
- Node
- Express
- GraphQL
